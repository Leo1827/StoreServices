# Zhowit

Zhowit is considered as a web page dedicated to the creation and administration of Physical Services with business-to-business and business-to-consumer models. All this through an attractive, dynamic and easy-to-use graphical interface.

Compiled with Vite

# How to start?

## 1. Downoal the project

the first step is download the project, you can do it with a git clone or download directly  of button de descargas.

## 2. Install dependencies
the next step is download dependencies and you can do it this with the next commands
-   npm run dev
-   npm run build
-   npm run preview
